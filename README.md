# RigidJudge
Judge of RigidOJ

## Features
### Local Judge
* Sandbox protection
* Special judge

### Remote Judge
* Proxy to remote Onlinejudge. (UVa POJ HDU ZOJ UVaLive SGU SPOJ AIZU Ural Codeforces Codeforces::Gym ...)

## Requirements
- JRE 7
- RabbitMQ 3.4
